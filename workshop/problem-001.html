An Impossible Diophantine Equation

The trick is simple: squares modulo 4 + infinite descent. Once you see it, the result becomes inevitable.

Problem

Find all integer solutions \((a,b,c)\) to

$$a^2 + b^2 + c^2 = a^2 b^2.$$

Key fact (mod 4)

For any integer \(n\), we have \(n^2 \equiv 0 \pmod 4\) if \(n\) is even, and \(n^2 \equiv 1 \pmod 4\) if \(n\) is odd. So a sum of squares is congruent to \(0,1,2,\) or \(3\) mod 4, and it is \(0 \pmod 4\) only when each square is \(0 \pmod 4\).

Step 1: show \(a\) and \(b\) must be even

Consider parities:

  • If \(a\) and \(b\) are both odd, then \(a^2b^2 \equiv 1 \pmod 4\). But \(a^2+b^2+c^2 \equiv 1+1+c^2 \equiv 2+c^2\), which is \(2\) or \(3\) mod 4, never \(1\). Contradiction.
  • If exactly one of \(a,b\) is odd, then \(a^2b^2\equiv 0 \pmod 4\). But \(a^2+b^2+c^2\equiv 1+0+c^2\), which is \(1\) or \(2\) mod 4, never \(0\). Contradiction.

Therefore, both \(a\) and \(b\) are even.

Step 2: show \(c\) is even

If \(a\) and \(b\) are even, then the right-hand side \(a^2b^2\) is divisible by 4, so the left-hand side must be divisible by 4 as well. But modulo 4, this forces \(a^2 \equiv b^2 \equiv c^2 \equiv 0\), hence \(c\) is even too.

Step 3: infinite descent

Write \(a=2a_1\), \(b=2b_1\), \(c=2c_1\). Substitute into the equation:

$$4(a_1^2+b_1^2+c_1^2)=16a_1^2b_1^2 \;\Rightarrow\; a_1^2+b_1^2+c_1^2=4a_1^2b_1^2.$$

Apply the same parity argument again: it forces \(a_1,b_1,c_1\) all even. Repeating forever, we get that \(a,b,c\) are divisible by \(2^n\) for every \(n\), which is only possible if \(a=b=c=0\).

Answer

The only integer solution is \(\boxed{(a,b,c)=(0,0,0)}\).

Takeaway: when an equation forces ‚Äúall variables are even‚Äù, try infinite descent. It is one of the cleanest ways to kill Diophantine equations.
