## Overview

The SOA FM Practice Tool is an interactive study tool designed to help candidates practice financial mathematics problems in a structured way.

The tool focuses on problem solving, progress tracking, and repeated exposure to key exam concepts such as interest theory, annuities, loans, bonds, yield rates, and immunization.

## Purpose

The goal of this project is to create a serious actuarial study environment, not just a static list of exercises. Each problem is connected to a topic, a solution, and eventually a larger learning path.

## Main Features

- Organized problem bank by topic
- Multiple-choice practice format
- Full solutions
- Progress tracking
- Count of uniquely solved problems
- Future integration with a larger SOA FM learning platform

This project is currently under development.
