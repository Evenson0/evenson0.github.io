## Overview

This section gathers the mathematical tools I build to explore, visualize, and explain mathematical ideas.

The goal is not only to write about mathematics, but also to create interactive objects that help users experiment with concepts directly. These tools are part of my broader effort to build a serious mathematical portfolio combining exposition, programming, and research-oriented thinking.

## Purpose

Mathematics is often easier to understand when we can manipulate examples, change parameters, and observe how structures evolve. My tools are designed with this idea in mind: turning abstract definitions and formulas into interactive experiences.

## Current and Planned Tools

### Random Walk Tool

An interactive tool for exploring random walks step by step. The goal is to visualize how randomness accumulates over time and how probabilistic behavior emerges from simple repeated rules.

### Binomial Tree Tool

A tool for visualizing binomial trees in option pricing. It shows how the tree evolves, how asset prices are generated, and how option values can be computed backward through the tree.

### SOA FM Practice Tool

An actuarial mathematics practice tool focused on Exam FM topics such as interest theory, annuities, loans, bonds, yield rates, and immunization.

### Future Mathematical Tools

Planned tools include visualizations for combinatorics, generating functions, stochastic processes, actuarial models, and financial mathematics.

## Long-Term Vision

The long-term goal is to build a library of mathematical tools that can support learning, research, and actuarial applications. Each tool will eventually be connected to a GitHub repository, technical documentation, and related articles.
