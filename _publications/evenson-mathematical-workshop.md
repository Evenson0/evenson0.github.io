## Description

*Evenson's Mathematical Workshop* is an evolving collection of mathematical problems and solutions written in the spirit of mathematical competitions and classical problem solving.

The goal of this project is to gather and preserve the problems that I solve and publish over time on my blog.

Rather than leaving each problem isolated in a blog post, this book brings them together into a single mathematical workshop where ideas, techniques, and elegant arguments can be refined and organized.

The collection covers several areas of elementary mathematics, including:

- combinatorics  
- algebra  
- number theory  
- olympiad-style problem solving  

Each problem is accompanied by a concise solution emphasizing the key mathematical idea.

The project is intended to grow gradually as new problems are solved and published.

You can also find many of these problems on my blog:

**https://evenson0.github.io**
